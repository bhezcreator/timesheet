| ------------: | --------------------- | :--------: | ------------------------------------------------- |
|     ✅ **400** | Bad Request           |     ⭐⭐⭐    | Requête mal formée                                |
|     ✅ **401** | Unauthorized          |    ⭐⭐⭐⭐    | Utilisateur non authentifié                       |
|     ✅ **403** | Forbidden             |    ⭐⭐⭐⭐⭐   | Accès refusé                                      |

|     ⭐ **405** | Method Not Allowed    |     ⭐⭐⭐    | Mauvaise méthode HTTP (GET/POST...)               |
|     ⭐ **408** | Request Timeout       |     ⭐⭐     | La requête du client a expiré                     |
| ⭐⭐⭐⭐⭐ **419** | Page Expired          |    ⭐⭐⭐⭐⭐   | Session CSRF expirée (très fréquent avec Laravel) |
|    ⭐⭐ **422** | Unprocessable Content |     ⭐⭐⭐    | Validation échouée (API principalement)           |
|   ⭐⭐⭐ **429** | Too Many Requests     |    ⭐⭐⭐⭐    | Limitation de débit (Rate Limiting)               |
|     ✅ **500** | Internal Server Error |    ⭐⭐⭐⭐⭐   | Erreur interne                                    |
|     ✅ **502** | Bad Gateway           |     ⭐⭐⭐    | Reverse Proxy                                     |
|     ✅ **503** | Service Unavailable   |    ⭐⭐⭐⭐    | Maintenance                                       |
|     ✅ **504** | Gateway Timeout       |     ⭐⭐⭐    | Timeout serveur                                   |
